DxE Colorado Events Calendar

Hey there! If you're one of the people responsible for creating and hosting DxE Colorado events, here's our event creation process:

  1. First, create a public Facebook event on our page with all of the event details and a full description.
  2. Second, create an event on the "DxE Colorado Calendar" Google Calendar with only this information: the event name, the event date and time, and a link to the Facebook event page in the "location" field.
